# Отчет по практическому занятию 3
## "Проектирование и публикация .NET-библиотеки с CI/CD"

### 1. Цели
- Освоить создание .NET-библиотеки с CI/CD
- Настроить автоматическую сборку, тестирование и публикацию в GitHub Packages

### 2. Ход выполнения
#### Структура проекта:
- Todo.Core (библиотека классов)
- Todo.Core.Tests (тесты xUnit)

#### Код библиотеки:
- Класс TodoItem (задача)
- Класс TodoList (список задач)

#### Конфигурация workflows:
- ci.yml (сборка и тестирование)
- publish-package.yml (публикация пакета)

#### Ссылка на опубликованный пакет:
https://github.com/YanaYana22/TodoLibrary/packages

### 3. Вывод
В ходе работы успешно создана .NET-библиотека, настроен CI/CD pipeline, пакет опубликован в GitHub Packages. Дополнительно реализована JSON-персистенция задач.
